Fluxus

A direct attach, inline current probe for an oscilloscope.

As my projects get more and more complex, I've been finding the desire for more testing capabilities. The problem is that I have a pretty small desk, so large external devices tend to be a pain.

This project is designed to give me an insight into various current waveforms whilst taking advantage of a traditional oscilloscope.

Thus the requirements were:
- Panel mount to oscilloscope
- Powered via USB C
- ~+- 10A measurement range
- -3dB BW of ~1MHz
- Bipolar current sensing

To achieve the panel mount nature, I ended up using a panel mount BNC in a very non-standard way
